Using the ATL editor

Using content/code assist

A content assist (or code assist) simplifies you writing ATL code.

  1. Pressing <Ctrl>+<Space> in a valid position of the cursor on a line of code will cause the ATL editor to show you a list of possible completions in a floating window if it finds valid candidates for this cursor position. Typing more characters will regenerate and thus affine the list. A simple click on a element of the list may show you its documentation or information ;
  2. Use the arrow keys or the mouse pointer to navigate through the possible completions ;
  3. Select an item in the list to complete the fragment by doing one of the following :
    • Selecting it and pressing Enter
    • Double-clicking it
