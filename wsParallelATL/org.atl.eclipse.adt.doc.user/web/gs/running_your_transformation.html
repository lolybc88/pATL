Running your programs

In this section, you will learn more about running ATL transformations in the Eclipse workbench.

When programming in ATL or when executing ATL is is best to switch to the ATL or the Debug perspective.

In both perspectives you can run and debug ATL programs. However, it goes without saying that debugging
should be performed in debug mode.

In run mode, the program executes, but the execution may not be suspended or examined.

To launch your ATL transformation, use the drop-down Run menu in the toolbar and select "Run....".

See ATL Configuration page to know more about the creation of launch configuration.

When your ATL configuration is finished, you can run the ATL transformation (Button "Run" in ATL Configuration).

For debugging switch to the Debug perspective (Window).

Your transformation is running until the terminated state appears in the debug view. To see the result, open the target model.
