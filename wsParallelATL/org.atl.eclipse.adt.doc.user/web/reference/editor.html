Highlight syntaxing coloration

Syntax specifies how ATL source code is rendered.

Option Description Default
Background color

System default : The default background color given by the operating system
Custom : A user defined background color

 System default
Foreground

The following ATL source fragments can be rendered with different color and styles (bold and italic) :

Brackets : ATL brackets, such as '('
Constants : Constant valuable keywords, such as 'true'
Identifiers : Names or simple names
Keywords : All ATL keywords
Enum literals : Enumeration literals starting with '#'
Numbers : Integers, floats, ...
Operators : Every operators
Others : Default ATL source code
Single-line comment : Comments starting with '--'
Strings : ATL strings and characters, surrounded by single quotes
Symbols : Symbols that partition the code, such as ';'
Types : Variables' types

 Default colors and styles
