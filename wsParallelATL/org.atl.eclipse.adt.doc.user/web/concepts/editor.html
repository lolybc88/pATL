ATL editor

The ATL editor provides you the abilities to edit ATL code with the aim of transforming a kind of file in an other one, such as XMI to text.

The ATL outline view associtated with the editor shows you the structure of the current edited file.

The editor includes the following features :

  • Syntax highlighting ;
  • Content/code assist ;
  • Integrated debugging features.

The most common way to invoke the ATL editor is to open an ATL file from the Navigator or Package explorer using pop-up menus or by clicking the file (single or double-click depending on the user preferences).
